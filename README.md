# Posts

This is the home of my posts.
Any updates to this repository automatically update my [website](https://hkamran.com) through the [workflow](.github/workflows/revalidate.yml).

## Structure

The [index file](./index.json) is where metadata for the posts is stored.
The Markdown sources are in the `posts` folder.

## Contributions

If you would like to update my posts, please [fork this repository](https://github.com/hkamran80/articles/fork), make your contributions, then submit a [pull request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests).
